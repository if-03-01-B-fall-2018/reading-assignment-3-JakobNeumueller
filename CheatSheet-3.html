Cheat Sheet 3

Html for Lists

Start tags for...

  • an ordered list: ol (creates an ordered list)
  • an unordered list: ul (creates an unordered list)
  • a description list: dl... to create; dt... for the item you want to describe; dd... the description
  • a list item: li (defines a list item)
  • a description term: dt
  • a description data: dd

Css for Lists

Make bullet a square: list-style-type: square/bullet

Make bullet an image named "my-bullet.png": list-style-image: url("my-bullet.png")

Make bullet an uppercase roman number: list-style-type: upper-roman

Set bullet position to be inside the list item's text flow

Set bullet position to be outside the list item's text flow
